# 🏋️ Exercises

One file of hands-on drills per milestone. Do them **after** reading the milestone's
file in [`notes/`](../notes/) and **before/while** studying its project repo.
Each drill ends with a *lesson to confirm* — say it in your own words before moving on.

| Milestone | Drills | Project they prepare you for |
|:-:|--------|------------------------------|
| 1 | [LLM fundamentals](./milestone-1-llm-fundamentals.md) — tokens & cost, temperature, tool calling, forced JSON | `structured-extractor` |
| 2 | [RAG](./milestone-2-rag.md) — retrieval, chunking, hybrid search, golden sets | `rag-knowledge-assistant` |
| 3 | [Agents & MCP](./milestone-3-agents-mcp.md) — ReAct loop, step budgets, adding tools, MCP | `agentic-workbench` |
| 4 | [Evals & guardrails](./milestone-4-evals-observability.md) — scorers, LLM-judge, attacks, tracing | `llm-eval-kit` |
| 5 | [Serving & fine-tuning](./milestone-5-serving-finetuning.md) — FastAPI, caching, rate limits, CI/CD, QLoRA | `rag-knowledge-assistant` + `lora-finetune-lab` |
| 6 | [Capstone](./milestone-6-capstone.md) — end-to-end trace, failure modes, the 3-minute pitch | `flagship-ai-platform` |

**Rules of thumb**
- Milestones 2–6 run **fully offline** (no API key). Milestone 1's drills use your
  `GEMINI_API_KEY` (tiny/free-tier usage); every milestone also has an optional live drill.
- Don't rush. One honest drill beats three skimmed ones.
